# Accumulate

Given the collection of strings
and an operation (such as "upcase"),
your code should be able to produce the transformed collection of strings.

```
"cat", "Dog", "b4t", "gO"
"CAT", "DOG", "B4T, "GO"
```

Test:

```
go test -v --bench . --benchmem
```
